# Ressources de marque SpoolyTracker

Images générées à partir des logos officiels SpoolyTracker, aux formats et
dimensions attendus par le dépôt [home-assistant/brands](https://github.com/home-assistant/brands).

| Fichier | Dimensions | Usage |
|---|---|---|
| `icon.png` | 256×256 | Icône (thème clair) |
| `icon@2x.png` | 512×512 | Icône HiDPI |
| `dark_icon.png` | 256×256 | Icône (thème sombre) |
| `dark_icon@2x.png` | 512×512 | Icône sombre HiDPI |
| `logo.png` | 512×107 | Logo horizontal (thème clair) |
| `logo@2x.png` | 1024×214 | Logo HiDPI |
| `dark_logo.png` | 512×107 | Logo horizontal (thème sombre) |
| `dark_logo@2x.png` | 1024×214 | Logo sombre HiDPI |

## Publication des icônes dans Home Assistant

Ces fichiers ne sont pas chargés automatiquement par une intégration custom :
Home Assistant lit les icônes depuis le dépôt central `home-assistant/brands`.
Pour que le logo SpoolyTracker apparaisse dans l'UI, il faut soumettre une PR à
ce dépôt sous `custom_integrations/spoolytracker/` avec `icon.png`, `icon@2x.png`
et éventuellement `dark_icon.png`, `logo.png`, `logo@2x.png`.

Ces fichiers sont fournis ici prêts à l'emploi pour cette soumission.
